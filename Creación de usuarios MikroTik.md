# Creación de usuarios MikroTik

---

# Procedimiento

Este es el paso a paso para la creación de usuarios que permiten el acceso a equipos MikroTik.

---

## 1. Acceder al portal de registro

Ingresar al siguiente enlace:

http://usermanager.eternet.cc:61380/user/signup


---

## 2. Completar el formulario

Una vez dentro del sitio, completar los siguientes campos:

- Email
- Teléfono (formato: 549...)
- Nombre
- Apellido
- Usuario
- Contraseña (mínimo 5 caracteres)


---

## 3. Confirmar creación del usuario

- Hacer clic en el botón **“Sign up”**
- Esperar la confirmación del sistema

<img width="296" height="394" alt="image" src="https://github.com/user-attachments/assets/3e11288b-a309-4cb7-b497-727619fc1a26" />

---

## 4. Validación del usuario

Si la creación fue exitosa:

- En unos minutos el usuario quedará habilitado
- Podrá acceder a equipos MikroTik mediante Winbox

- Winbox:
https://github.com/AbutMatias/Inducci-n-Atencion-al-Cliente-/blob/main/Winbox.md

---

# Cambio de contraseña

Para modificar la contraseña:

- Ingresar nuevamente a:

http://usermanager.eternet.cc:61380/user/signup

- Utilizar el formulario correspondiente para actualización de credenciales
