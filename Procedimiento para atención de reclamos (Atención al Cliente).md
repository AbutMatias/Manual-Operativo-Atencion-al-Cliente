# Introducción

En esta guía se detallan los pasos a seguir cuando ingresa un reclamo por Botmaker, llamado o correo.

---

# Herramientas

Disponemos de 2 plataformas principales para registro y comunicación interna:

---

## SSAK

- Sistema principal de base de datos de clientes
- Se utiliza para:
  - Búsqueda de clientes
  - Carga de reclamos
  - Seguimiento interno
---

## GitHub

- Plataforma oficial de comunicación interna de Eternet
- Se utiliza para:
  - Derivaciones entre áreas
  - Registro detallado de reclamos
  - Adjuntar evidencia (capturas, logs, etc.)
  - Seguimiento de casos

# Procedimiento

El flujo completo se divide en 5 pasos:

---

## 1. Recepción del contacto

- Puede ingresar por:
  - Botmaker
  - Llamado telefónico
  - Correo electrónico
  - Issue

Importante:
- Responder rápidamente para confirmar recepción del caso

---

## 2. Carga del reclamo

- Verificar en SSAK si el cliente ya tiene un reclamo activo
- Si no existe:
  - Realizar la carga del reclamo

Instructivo:
https://github.com/AbutMatias/Inducci-n-Atencion-al-Cliente-/blob/main/Carga%20de%20reclamos.md

---

## 3. Diagnóstico del reclamo

- Agregar observaciones en SSAK
- Información clave:
  - Detalle del problema
  - Pruebas realizadas
  - Comportamiento del servicio

Importante:
- Estas notas pueden ser utilizadas por otros técnicos o sectores
- Deben ser claras y completas

---

## 4. Actualización de contacto

Antes de cerrar el contacto con el cliente:

- Actualizar:
  - Número de WhatsApp principal
  - Correo electrónico vigente

Obligatorio si hubo cambios detectados

---

## 5. Resolución o derivación

Una vez obtenido el diagnóstico final:

---

### Si el reclamo se resuelve en el sector

- Cerrar reclamo en SSAK
- Detallar:
  - Causa del problema
  - Solución aplicada



---

### Si se deriva a otro sector

Se debe:

1. Realizar derivación en SSAK al sector correspondiente  
2. Crear issue en GitHub usando la plantilla adecuada  
   https://github.com/Eternet/Reclamos/issues/new/choose
3. Asignar etiquetas correspondientes  
4. Vincular el issue al sector responsable
