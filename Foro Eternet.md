# Foro Eternet

## Introducción

El Foro Eternet es una plataforma que actualmente se encuentra en desuso, pero aún contiene información útil para consultas específicas.

En el foro se almacenaban:

- Diagramas de red de clientes.
- Fotografías de instalaciones.
- Documentación técnica del área de Redes.
- Relevamientos y registros históricos.

Aunque ya no se utiliza activamente, puede resultar útil para obtener información de clientes o instalaciones antiguas.

---

# Acceso al Foro

1. Abrir un navegador web.
2. Ingresar a la siguiente dirección:

   https://foro.eternet.cc

3. Iniciar sesión con las credenciales asignadas.

> Las credenciales fueron enviadas por correo electrónico a todos los integrantes del área de Atención al Cliente (diciembre de 2022).

---

# Búsqueda de Información

Para localizar información:

1. Ingresar al foro.
2. Utilizar el buscador ubicado en la parte superior.
3. Escribir palabras clave relacionadas con:
   - Nombre del cliente.
   - Dirección.
   - Número de cliente.
   - Localidad.
   - Equipo o dispositivo.
4. Revisar los resultados obtenidos.

---

# Cuándo Consultar el Foro

Se recomienda buscar en el foro cuando:

- Se necesita información histórica de una instalación.
- Se requieren diagramas de red antiguos.
- Se buscan fotografías de equipos o montajes.
- Se necesita documentación técnica que no se encuentra en otros sistemas.
- Se requiere información relevada anteriormente por el área de Redes.

---

# Resultado Esperado

Si existe documentación previa del cliente o instalación, el foro puede contener:

- Fotografías.
- Diagramas.
- Configuraciones.
- Notas técnicas.
- Relevamientos realizados por personal de Eternet.

Si no se encuentran resultados, continuar el diagnóstico utilizando las herramientas y sistemas actualmente vigentes.
