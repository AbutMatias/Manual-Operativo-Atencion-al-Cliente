# Glosario básico de Atención al Cliente

Este glosario reúne los términos más utilizados en diagnóstico y soporte técnico dentro de Eternet.

---

| Término | Descripción |
|--------|-------------|
| Host | Router principal del cliente. Contiene la configuración de la IP pública. |
| CPE | Equipo intermedio entre la red y el host. Puede ser ONU, equipo wireless o router en modo bridge. |
| Bridge | Equipo en modo puente que permite extender la gestión hasta el host o la última milla. |
| Ethernet | Interfaz física de red por cable RJ45. |
| LAN | Red de área local. |
| WLAN | Red inalámbrica de área local. |
| SFP | Interfaz para conexión de fibra óptica. |
| Transceiver | Módulo que se conecta en puertos SFP para fibra óptica. |
| Conector rápido de fibra | Conector utilizado en redes FTTH para instalaciones de alta densidad. |
| FTTH | “Fiber To The Home” (fibra hasta el hogar). |
| Equipo Wireless | Dispositivo de conexión inalámbrica. Puede ser Access Point o Station. |
| Access Point (AP) | Equipo que brinda señal a múltiples estaciones. |
| Station | Equipo cliente que se conecta a un Access Point. |
| P2P | Enlace punto a punto entre dos equipos. |
| P2MP | Enlace punto a multipunto (un AP hacia varios clientes). |
| Subred | Conjunto de dispositivos conectados dentro de una misma red lógica o física. |
| Hotspot | Equipo o sistema que valida clientes dentro de una subred. |
| BGP | Protocolo de enrutamiento dinámico para redundancia entre redes. |
| DHCP | Protocolo que asigna automáticamente IP y configuración de red a los dispositivos. |
| DNS | Sistema que traduce nombres de dominio a direcciones IP. |
| Ping | Prueba de conectividad que mide latencia en milisegundos. |
| MAC Address | Identificador físico único de un dispositivo de red. |
| Neighbors | Lista de dispositivos MikroTik detectados en la red (en desuso). |
| Firmware | Software interno de bajo nivel de un dispositivo. |
| Log | Registro de eventos del sistema o equipo de red. |
| QoS | Calidad de servicio. Permite priorizar tráfico de red. |
| Queue | Colas de tráfico usadas para limitar o priorizar ancho de banda. |
| Reboot | Reinicio del dispositivo. |
| Reset | Restablecimiento a valores de fábrica. |
| Bandwidth test | Prueba de ancho de banda entre equipos en MikroTik. |
| Speedtest | Prueba de velocidad de conexión a un servidor. |
| Graphing | Gráficos de tráfico de red en tiempo real o histórico. |
| RoMON | Protocolo MikroTik para acceder a equipos sin IP directa. |
| Traceroute | Herramienta que muestra la ruta hacia un destino en la red. |
| Telnet | Protocolo de acceso remoto por línea de comandos. |
