# Solicitud de Baja de Servicios

# Introducción

A continuación se describe el procedimiento mediante el cual se gestionan las solicitudes de baja.

Este proceso centraliza la información y establece un sistema de seguimiento que garantiza que cada gestión sea monitoreada de manera efectiva, asegurando que todas las solicitudes sean atendidas.

De esta manera, se optimiza la gestión y se facilita el cumplimiento de los plazos y requisitos.

---

# Recepción de la Solicitud de Baja

Cuando un sector recibe una solicitud de baja por parte de un cliente debe:

1. Acceder al formulario de solicitud de baja.
2. Completar todos los campos requeridos.
3. Registrar correctamente el motivo por el cual el cliente solicita la baja.
4. Verificar que la información cargada sea correcta antes de enviar el formulario.

# Gestión desde Atención al Cliente

> [!IMPORTANT]
> Cuando la solicitud de baja es recibida por el área de Atención al Cliente, se debe realizar siempre una instancia de retención antes de continuar con el proceso.

## Retención Nivel 1

Se aplica la metodología de retención de primer nivel:

- Escucha activa.
- Identificación del motivo de baja.
- Comunicación de beneficios del servicio.
- Oferta de alternativas disponibles.

### Resultados posibles

#### ✅ Cliente retenido

Si la retención resulta exitosa:

- Seleccionar la opción **"Retenida 1° Nivel"** en el formulario.
- Registrar los detalles correspondientes.

#### ❌ Cliente no retenido

Si no es posible retener al cliente:

- Seleccionar la opción **"Gestiones Especiales"**.
- Continuar con el flujo de derivación correspondiente.

---

# Gestiones Especiales

Cuando una solicitud es derivada a Gestiones Especiales:

- La gestión es recepcionada por el responsable designado del sector.
- Se analiza el caso particular.
- Se evalúan herramientas adicionales de retención.
- Se realiza seguimiento hasta obtener una resolución.

Actualmente el responsable de recepcionar estas solicitudes es:

**Federico Barrio**

# Beneficios del Proceso

La información recopilada mediante el formulario permite obtener métricas y generar acciones de mejora continua.

Entre los principales indicadores que se pueden analizar se encuentran:

- 📊 Medición de intención de baja por localidad.
- 📊 Medición de motivos de baja por localidad.
- 📊 Porcentaje de retención por localidad.
- 📊 Porcentaje de retención según motivo.
- 📊 Seguimiento de herramientas de retención utilizadas.
- 📊 Medición de competencia.
- 📊 Evolución de solicitudes de baja en el tiempo.
- 📊 Efectividad de las acciones comerciales implementadas.

---

# Consideraciones Importantes

> [!IMPORTANT]
> Toda solicitud de baja debe registrarse mediante el formulario correspondiente.

> [!IMPORTANT]
> No debe procesarse una baja sin haber realizado previamente la instancia de retención correspondiente.

> [!NOTE]
> El correcto registro de los motivos de baja es fundamental para la generación de estadísticas y la toma de decisiones comerciales.
