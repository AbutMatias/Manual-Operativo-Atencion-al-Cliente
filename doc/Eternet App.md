# Eternet App

Desde la [App Eternet](https://app.eternet.com.ar/) se gestionan los datos de los clientes.  
Esta fuente de datos es la misma que se utiliza para el sitio web, Botmaker, recuperación de clave de Webmail, etc.

Aunque contamos con un ABM en SSAK, este no se actualiza de manera inmediata.  
Por ello, para consultar datos de un cliente siempre se debe utilizar la App.

# Opciones disponibles

## 1. Agregar o eliminar contacto

<img width="1770" height="869" alt="image" src="https://github.com/user-attachments/assets/1b5a46d4-1367-4d90-abbb-e26af06bf48a" />


- Ir a la solapa **Medios de contacto**
- Se puede:
  - Agregar contacto
  - Eliminar contacto

Importante:
- Solo se puede modificar si el contacto **NO está verificado**
- Si está verificado, el cliente debe gestionarlo desde la web:
  https://eternet.com.ar/oficina-virtual/mis-medios-de-contacto
