# Hosting - Dominio

## ¿Qué es Hosting?

Un hosting o alojamiento web es un servicio que permite almacenar sitios web, aplicaciones, bases de datos y archivos relacionados en servidores conectados a Internet. Gracias a este servicio, cualquier usuario puede acceder a una página web desde cualquier lugar del mundo.

En términos simples, el hosting es el espacio donde se guardan todos los archivos necesarios para que un sitio web funcione.

## Diferencia entre Hosting y Dominio

### Hosting
Es el lugar donde se almacenan los archivos de un sitio web. Una empresa proveedora de hosting alquila espacio en sus servidores para alojar páginas web y servicios asociados.

### Dominio
Es la dirección única que permite acceder a un sitio web, por ejemplo:

`www.miempresa.com.ar`

Funciona de manera similar a la dirección de una vivienda: identifica de forma única un sitio dentro de Internet.

## ¿Qué es un Email Hosting?

El Email Hosting es un servicio de alojamiento de correo electrónico que permite utilizar cuentas de correo personalizadas con el dominio de la empresa, por ejemplo:

`ventas@miempresa.com.ar`

Además de almacenar los correos electrónicos, permite administrar cuentas, buzones y configuraciones asociadas.

# Proceso de Venta

Existen dos tipos de solicitudes:

- **Alta de nuevo dominio**
- **Transferencia de dominio existente** (cambio de hosting)

En caso de solicitar un dominio nuevo, se debe verificar previamente su disponibilidad en NIC Argentina:

🔗 https://nic.ar/buscar-dominio

> [!NOTE]
> El servicio de Hosting no está incluido dentro de los planes de Internet y se ofrece únicamente a solicitud del cliente.

# Gestión de Carga

Una vez aprobada la propuesta por parte del cliente, se debe generar una Issue con las tareas correspondientes para cada sector involucrado.

## Servicio Técnico y Servidores

Responsables de:

- Gestionar el alta técnica del servicio.
- Configurar el hosting.
- Configurar cuentas de correo.
- Contactar al cliente para validar el correcto funcionamiento.

## Administración

Responsable de:

- Facturar el servicio según el presupuesto aprobado.
- Registrar los cargos correspondientes una vez finalizadas las tareas técnicas.

## Comercial

Responsable de:

- Dar de alta el ítem **Hosting**.
- Incorporar módulos adicionales de cuentas de correo cuando corresponda.

---

# Gestión de Baja

Ante una solicitud de baja:

## Servicio Técnico y Servidores

Responsables de:

- Eliminar los servicios asociados.
- Gestionar la baja técnica del hosting y correos.

## Comercial

Responsable de:

- Dar de baja el ítem **Hosting** en el sistema.

> [!IMPORTANT]
> Informar siempre al cliente que la baja del servicio implica la eliminación definitiva de los recursos asociados, incluyendo:
>
> - Sitio web.
> - Cuentas de correo.
> - Correos electrónicos almacenados.
> - Archivos alojados en el hosting.
>
> Una vez procesada la baja, la información no podrá recuperarse.
