# 📞 Telefonía IP (VoIP - Protocolo de telefonía por Internet)

## ¿Qué es la Telefonía IP?

La Telefonía IP es una tecnología que utiliza el **protocolo IP** para el intercambio de voz, fax y otros tipos de comunicación.

A diferencia de la telefonía tradicional, que utiliza la Red Telefónica Pública Conmutada (PSTN), la Telefonía IP transmite la información en forma de paquetes a través de redes como:

- Redes LAN (red local)
- Internet

Esto permite reducir o eliminar costos asociados a la red telefónica tradicional.

# 🚫 Procedimiento de baja del servicio

Para dar de baja el servicio de Telefonía IP:

1. Seleccionar el cliente  
2. Ir a **Ver Detalles → Clientes → Telefonía**  
3. Desmarcar la opción:  
   - `El cliente tiene Servicio de Telefonía`
4. Ir a **Datos Personales**
5. Guardar los cambios

# 📦 Recuperación de equipos

Para gestionar la recuperación de equipos:

1. Ir a:  
   **Servicios → Servicios Próximos → Servicios Contratados → Baja de Servicio**

2. Completar:
   - Motivo de la baja
   - Observaciones
   - Seleccionar “Servicio Próximo”

# 🔧 Procedimiento de alta del servicio

El alta del servicio es realizada por el Área de Servicio Técnico e incluye:

- Configuración de equipos VoIP
- Alta de ID / número corto en la central telefónica
- Monitoreo del servicio ante reclamos
- Validación de funcionamiento en central y equipo del cliente
