# Baja del Servicio Eternet Plus

El proceso de baja de un servicio **Eternet Plus** se realiza seleccionando la opción:

```text
Baja de Servicio WIFI (con Equipo a Retirar)
```

El equipo entregado al cliente permanece en el domicilio, por lo que es necesaria la intervención del sector de **Retiros** para coordinar la recuperación del router.

---

# 1. Carga por Sistema

Para realizar la baja:

1. Ingresar a **SSAK47**.
2. Acceder al menú **Atención al Cliente**.
3. Buscar al cliente correspondiente.
4. Ingresar al **AMB** desde la solapa **Clientes**.

## Selección de la opción de baja

Una vez dentro del cliente:

1. Ubicarse sobre el servicio **WIFI** correspondiente.
2. Hacer clic derecho sobre el servicio.
3. Seleccionar una de las opciones disponibles.

El sistema mostrará las siguientes alternativas:

- Baja de Servicio WIFI (con recibo de equipos)
- Baja de Servicio WIFI (con Equipo a Retirar)

<img width="1003" height="614" alt="image" src="https://github.com/user-attachments/assets/02ec7714-f85e-4ce9-ae81-bf991031f53f" />

## Opción Correcta

> [!IMPORTANT]
> Siempre debe seleccionarse la opción:
>
> ```text
> Baja de Servicio WIFI (con Equipo a Retirar)
> ```

Una vez realizada la operación, SSAK47 confirmará la baja del servicio y dejará los equipos pendientes de retiro.

# 2. Carga de Issue

Luego de realizar la baja administrativa, es obligatorio generar la Issue correspondiente para que el servicio Eternet Plus sea efectivamente deshabilitado.

> [!WARNING]
> Si este paso no se realiza, el cliente continuará teniendo activo el servicio Eternet Plus aunque la baja administrativa ya se haya procesado.

---

## Formulario a utilizar

Utilizar la siguiente Issue:

```text
Reactivación / Suspensión Eternet Plus
```

Completar toda la información requerida y enviarla al área correspondiente.

[Formulario de Issue](https://github.com/Eternet/Backoffice.Tecnico/issues/new?template=10%20-%20Reactivacion_Suspension_ServicioEternetPlus.yml)

---

# Resumen del Procedimiento

1. Ingresar al cliente en SSAK47.
2. Acceder al servicio WIFI.
3. Seleccionar:
   - Baja de Servicio WIFI (con Equipo a Retirar)
4. Confirmar la baja.
5. Generar la Issue de Reactivación / Suspensión Eternet Plus.
6. Verificar que la gestión quede correctamente derivada.

> [!NOTE]
> La baja administrativa por sí sola no deshabilita el servicio Eternet Plus. La Issue es obligatoria para completar el proceso.
